# Calculator Project

A simple web-based calculator built with HTML, CSS, and JavaScript.  
This project demonstrates basic DOM manipulation and arithmetic operations.

---

## Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, division  
- Clear the display with a single button  
- Dynamic update of calculation as you type  
- Simple and responsive layout  

---

## Technologies Used

- **HTML** for structure  
- **CSS** for styling (`calc-style.css`)  
- **JavaScript** for interactivity (`functinality.js`)
